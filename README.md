Python Altersrechner
Ein einfaches Python-Tool, das das genaue Alter basierend auf dem Geburtsdatum des Benutzers berechnet. Dieses Projekt wurde erstellt, um den professionellen Git- und GitHub-Workflow zu üben.

Projektbeschreibung
Das Programm verwendet die integrierte datetime-Bibliothek von Python, um das aktuelle Datum abzurufen und es mit den Benutzereingaben (Jahr, Monat, Tag) zu vergleichen.

Technologien
Sprache: Python 3.x

Versionsverwaltung: Git & GitHub

Editor: VS Code

Installation und Ausführung
Führen Sie den folgenden Befehl im Terminal aus, um das Programm zu starten:
python dateeingabe.py
